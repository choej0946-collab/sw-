# 장바구니 프로젝트

## 프로젝트 개요

- 장바구니 프로젝트 구현
- 최지우
- 사용자가 상품을 장바구니에 담고, 수량 변경 및 총 금액 확인이 가능한 쇼핑몰 핵심 기능 구현

## 개발 환경

React 19.2.0, jsx와 css를 주로 이용해서 만들었습니다.

## 설치 및 실행 방법

nodejs 설치 
npm create-react-app . 
npm start

## 컴포넌트 구조

CartPage.jsx	전체 장바구니 페이지의 메인 컨테이너
CartItem.jsx	개별 상품 정보(이미지, 이름, 단가, 수량, 총액) 표시
QuantitySelector.jsx	상품 수량을 + / - 버튼으로 조정
CartSummary.jsx	총 상품 금액, 배송비, 최종 결제 금액 표시
FreeShippingBanner.jsx	무료배송 조건 안내 (“10만원 이상 무료배송”)
CheckoutButton.jsx	결제 페이지로 이동 버튼

## 테스트 방법

npm test

주요 테스트 항목:

수량 증가/감소 시 총 금액 계산이 정확히 반영되는지

10만원 이상일 때 배송비가 자동으로 0원으로 변경되는지

상품 삭제 시 합계 금액이 즉시 갱신되는지


## 유의사항 및 알려진 이슈

배송비 배너의 텍스트가 모바일 해상도에서 줄바꿈 이슈 있음

합계 금액 영역의 시각적 강조 부족 (차후 디자인 보완 예정)

향후 기능 확장 계획: 쿠폰 할인 적용 기능, 로그인 상태 연동, 선택 상품만 결제 기능
