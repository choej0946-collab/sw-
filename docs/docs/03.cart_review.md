# 장바구니 페이지 리뷰

## 고객사 요구사항
>●	금액의 정확성 중요함. 상품의 가격, 수량, 수량에 따른 총 금액, 배송비까지 한 눈에 볼 수 있게 해주기. 10만원 이상 구매 시 배송비 무료.

## 요구사항에 따른 기능 동작 확인

> 컴포넌트	주요 기능	확인 내용	결과	비고
CartItem (상품 카드)	상품명, 단가, 수량, 소계 표시	- 단가와 수량이 정확히 표시됨
- 수량 변경 시 소계 자동 계산	
QuantitySelector (수량 조절 버튼)	수량 증가/감소	- “+ / -” 버튼 클릭 시 수량 변경
- 1 미만으로 감소 불가
- 변경 즉시 금액 갱신
CartSummary (합계/배송비 영역)	총 상품 금액, 배송비, 최종 결제 금액 표시	- 모든 값 실시간 반영
- 배송비 3,000원 기본 적용
- 총액 ≥ 100,000원 시 배송비 0원 처리
FreeShippingBanner (배송비 안내 배너)	무료배송 조건 안내	- 조건 충족 시 “무료배송 적용” 표시
- 조건 미충족 시 남은 금액 안내 (“○○원 더 구매 시 무료배송”)
CartPageLayout (전체 페이지 레이아웃)	금액 정보 가시성 확보	- 각 섹션이 시각적으로 구분
- 합계 영역이 하단 고정되어 항상 표시됨	⚠️ 개선 권장	합계 금액 강조 색상 추가
CheckoutButton (결제 버튼)	결제 페이지 이동	- 클릭 시 결제 페이지로 정상 이동
- 비활성 상태 조건 없음
