# 결제 모듈 프로젝트

## 프로젝트 개요

- 결제 모듈 만들기
- 카드를 등록하고 결제할 수 있게 하는 컴포넌트
- 최지우

## 개발 환경

React 19.2.0, jsx와 cs를 주로 이용해서 만들었습니다.

## 설치 및 실행 방법

nodejs 설치
npm create-react-app .
npm start

## 컴포넌트 구조

주요 컴포넌트들의 구조와 역할

## 테스트 방법

아직 테스트를 못 했습니다.

## 유의사항 및 알려진 이슈

카드 등록할 때 이름도 함께 작성할 수 있도록 만들고 싶습니다.

## 추가 항목

유튜버 Dear Programmer의 강의 보면서 figma 디자인을 react 코드로 바꾸는 방법을 참고했습니다.
