# 쇼핑몰 연동

## 프로젝트 개요

- 본 프로젝트는 제품 목록, 상세 페이지, 장바구니, 결제 페이지가 연동된 쇼핑몰 기능 구현을 목표로 합니다.
- 주요 기능
제품 목록 조회
제품 상세 페이지 진입
장바구니 담기 및 수량 조절
장바구니 상태 유지(페이지 이동 시 반영)
결제하기 및 결제 완료 페이지 이동

- 참여자: 최지우

## 개발 환경

React 17.0 / JavaScript ES6
JSX / CSS
Node.js 18+
react-router-dom

## 설치 및 실행 방법

## 테스트 방법

1. 제품 상세 페이지 이동 테스트

제품 카드를 클릭 → 상세 페이지로 이동되는지 확인

2. 장바구니 수량 반영 테스트

메인 페이지 또는 상세 페이지에서 제품 “장바구니 담기” 클릭

헤더의 장바구니 아이콘 숫자가 증가하는지 확인

3. 결제 페이지 이동 테스트

장바구니 페이지 → “결제하기” 클릭

결제 완료 페이지가 정상적으로 보이는지 확인

4. 장바구니 수량 유지 테스트

제품 상세 페이지 또는 장바구니 페이지에서 수량 변경

메인 페이지 등 다른 페이지로 이동

다시 돌아왔을 때 변경된 수량이 그대로 유지되는지 확인

```
npm test
```

## 유의사항 및 알려진 이슈

장바구니 상태 저장 방식이 로컬 스토리지 기반이므로, 로그인 기능 없이 사용자별 분리가 되지 않음

새로고침 시 UI는 유지되지만 서버 연동 API가 없는 상태에서는 재적재가 어려움

네트워크 요청이 없는 구조라 실제 결제 기능은 구현되어 있지 않음
