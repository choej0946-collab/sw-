# 상품 목록 페이지 프로젝트

## 프로젝트 개요

- 상품 목록 페이지 모바일 앱 만들기
- 상품을 선택하여 장바구니에 추가할 수 있는 기능을 제공하는 앱을 개발
- 혼자 만들었습니다.


## 개발 환경

React 19.2.0

## 설치 및 실행 방법

nodejs 설치
npm create-react-app .
npm start

## 컴포넌트 구조

- Header : 헤더
- ProductList: 상품 목록 페이지
- ProductCard: 상품 카드 컴포넌트
- CartPage : 장바구니 페이지
- CartItem : 장바구니 아이템 컴포넌트

## 테스트 방법

아직 테스트를 못 했습니다.

## 유의사항 및 알려진 이슈

tailwind로 UI를 다시 만들고 싶다.

## 추가 항목

지피티를 사용해서 고칠 부분을 고치고 있습니다.
